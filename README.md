> [hyper](https://github.com/hyperium/hyper/) binding for [libgit2 bindings for Rust](https://github.com/rust-lang/git2-rs/)

## Rust version requirements

1.53.0

## How to build

```shell
$ git clone https://github.com/henry40408/git2-hyper
$ cd git2-hyper
$ cargo build
```
